🔍 Analisis de patrones AOB
// Patron AOB encontrado en memory.dump
48 8B 05 ?? ?? ?? ?? 48 85 C0 74 0A
48 8B 40 10 48 85 C0 74 03
FF 60 48 CC 90 90 90
IA
El patron 48 8B 05 ?? ?? ?? ?? es un mov rax, [rip+offset] tipico de acceso a variable global en x64. Los bytes 48 85 C0 74 0A son un test rax, rax / je que verifica si el puntero es nulo. Esto suele ser una estructura global como g_pGameManager o similar.
🧠 Modificacion de memoria
// C++ - WriteProcessMemory example
DWORD pid = GetProcessIdByName("game.exe");
HANDLE hProc = OpenProcess(PROCESS_ALL_ACCESS, FALSE, pid);
int newHP = 99999;
WriteProcessMemory(hProc, addr, &newHP, sizeof(newHP), &written);
IA
Codigo completo para modificar memoria de un proceso. Incluye GetProcessIdByName, OpenProcess con permisos totales y WriteProcessMemory. El codigo busca el proceso por nombre, obtiene su PID, abre un handle y escribe el nuevo valor en la direccion especificada.
💉 Inyeccion DLL
// C++ - DLL Injection via CreateRemoteThread
LPVOID remoteMem = VirtualAllocEx(hProc, NULL, dllPath.Size(), MEM_COMMIT, PAGE_READWRITE);
WriteProcessMemory(hProc, remoteMem, dllPath, dllPath.Size(), &written);
LPVOID loadLib = GetProcAddress(GetModuleHandle("kernel32.dll"), "LoadLibraryA");
CreateRemoteThread(hProc, NULL, 0, loadLib, remoteMem, 0, NULL);
IA
Inyeccion DLL clasica via CreateRemoteThread. Reserva memoria en el proceso remoto, escribe la ruta de la DLL, obtiene la direccion de LoadLibraryA y crea un hilo remoto que la carga. Incluye manejo de errores y espera de hilo.

Como instalar

Dos formas de usar MemeCraft Code en tu PC

📦

Version Portable

  1. Descarga MemeCraft-Code-1.0.23-portable.exe
  2. Guardalo en cualquier carpeta (escritorio, USB, etc.)
  3. Haz doble clic en el .exe para ejecutar
  4. No necesita instalacion, no modifica el registro
  5. Para actualizar: descarga la nueva version y reemplaza el .exe
⚙️

Version Instalador

  1. Descarga MemeCraft-Code-1.0.23-Setup.exe
  2. Ejecuta el instalador y sigue los pasos
  3. Elige carpeta de instalacion (o deja la default)
  4. Se crea acceso directo en el escritorio y menu inicio
  5. Se actualiza automaticamente cuando hay nuevas versiones
🔑

Primer uso

  1. Al abrir el app, escribe un mensaje y presiona Enter
  2. Si las claves integradas fallan, haz clic en API Key (arriba a la derecha)
  3. Presiona Probar para verificar si la API responde
  4. Si no funciona, genera tu key gratis en build.nvidia.com
  5. Pega tu API Key y guarda — listo para usar
⌨️

Atajos rapidos

  1. Enter — Enviar mensaje
  2. Ctrl+1 a Ctrl+0 — Cambiar modelo
  3. Ctrl+L — Limpiar chat
  4. Ctrl+E — Exportar a TXT
  5. Ctrl+F — Buscar en el chat
  6. Ctrl+Z — Deshacer ultimo envio
  7. ? — Ver todos los atajos

Preguntas frecuentes

Todo lo que necesitas saber sobre MemeCraft Code

¿Es completamente gratis?

Si. El app es open source y las claves integradas de NVIDIA NIM son gratuitas. Si se agotan, puedes generar tu propia API Key gratis en build.nvidia.com.

¿Que modelos incluye?

13 modelos: Llama 3.1 8B, DeepSeek V4 Flash, DeepSeek V4 Pro, Llama 3.3 70B, Qwen 3.5 397B, Nemotron Super 49B, Gemma 3, Llama Vision, Mixtral 8x7B, MiniMax M2.7, Poolside Laguna, GLM 5.2 y Kimi K2.6.

¿Puedo usar mi propio proveedor de IA?

Si. El boton 🔑 API Key te permite configurar cualquier URL base y API key compatible con OpenAI (OpenAI, Anthropic, Groq, Together, Azure, etc).

¿Como me actualizo?

El instalador (Setup.exe) se actualiza automaticamente via GitHub. La version portable necesitas descargar la nueva version manualmente.

¿Donde reporto bugs o sugiero mejoras?

En el repositorio de GitHub: github.com/bddjf00-cell/MemecraftIA/issues
